- vi: visual
- vim: Vi IMproved
- vi操作模式
    - command mode
    - insert mode
    - last line mode
- vi编辑命令
    - 移动光标
        - h, j, k, l, 左下上右
        - \<Ctrl> + \<b> 上翻页， \<Ctrl> + \<f> 下翻页
        - 0: 移至行首; $: 移至行末; w: 移至下个单词首
        - G: 移至末行行首; nG: 移至第n行
    - 删除
        - 字符删除: x, nx
        - 行删除: dd, ndd
    - 复制
        - 字符复制: y, yw
        - 行复制: yy, nyy
    - 粘贴: p
    - 查找字符串
        - /关键字: 向下查找
        - ?关键字: 向上查找
    - 撤销或重复操作
        - u: 撤销
        - .: 重复
- 末行模式操作
    - 列出行号: set nu
    - 跳至某行: number
    - 替换字符串: 范围/字符串1/字符串2/g
